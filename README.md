## 🤖 AI-Powered Video Extraction

Проект підтримує інтеграцію з OpenAI для розумнішої екстракції відео:

### Налаштування OpenAI

1. Отримайте API ключ від [OpenAI](https://platform.openai.com/api-keys)
2. Додайте змінну середовища:
   ```bash
   export OPENAI_API_KEY="your-api-key-here"
   ```
3. Для Render додайте змінну в Dashboard → Environment

### Що робить AI:

- **Розумний аналіз HTML** — знаходить плеєри та джерела відео
- **Інтерактивна взаємодія** — розуміє як натискати кнопки для завантаження відео
- **Покращена детекція** — розпізнає складні структури плеєрів

Без API ключа працює звичайна екстракція.
